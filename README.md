# wineup
Baixa e atualiza versões do WINE e PROTON, com interface em Zenity
* Compativel com:
  * WINE-GE
  * WINE-Lutris
  * WINE-TKG
  * Proton-GE
  * Proton-TKG
  * Luxtorpeda-DEV
 
## Dependencias:
 * lynx
 * zenity
 * aria2

## Modo de Uso:
### Para baixar uma versão especifica use a opção "-d"
$ wineup -d
